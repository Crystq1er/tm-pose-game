# 게임 규칙: Fruit Catcher (Infinite Mode)

## 🎮 게임 제목
**Fruit Catcher: Survival**

## 📝 게임 설명
폭탄을 피해 최대한 오랫동안 살아남으며 과일을 모으는 서바이벌 게임입니다!
생명(하트) 3개가 주어지며, 폭탄에 3번 맞으면 게임이 종료됩니다.
방어막 아이템을 활용하여 위기를 모면하세요!

## 🎯 게임 목표
- **무제한 시간**: 게임 오버 전까지 최대한 높은 점수를 획득하세요.
- **생존**: 폭탄을 피하고 생명을 유지하세요.

## 🕹️ 게임 규칙

### 기본 규칙
1. **시작**: Start 버튼으로 시작.
2. **생명 시스템**: ❤️❤️❤️ (3개)로 시작합니다.
3. **게임 오버**: 폭탄에 맞아 생명이 0이 되면 종료됩니다.

### 아이템 목록 (업데이트됨)
| 아이콘 | 이름 | 점수 | 효과 | 등장 확률 |
|:---:|:---:|:---:|:---|:---|
| 🍎 | **사과** | +100점 | 기본 점수 | 보통 |
| 🍊 | **오렌지** | +200점 | 높은 점수 | 보통 |
| 🍇 | **포도** | +300점 | 아주 높은 점수 | 낮음 |
| 🛡️ | **방어막** | 0점 | 폭탄 1회 방어 (중복 불가) | 낮음 |
| 🎁 | **선물상자** | +1000점 | 대량 점수 지급 | **매우 낮음** |
| 💣 | **폭탄** | 0점 | **생명 -1 감소** (방어막 있으면 방어) | **높음** |

> **선물상자**: 매우 드물게 등장하지만 높은 점수를 줍니다.
> **폭탄**: 자주 등장하니 주의하세요!

## 🎨 조작법 (동일)
- **Left**: 왼쪽 이동
- **Center**: 중앙 이동
- **Right**: 오른쪽 이동

## 📊 난이도
- **시간 제한 없음**: 무한히 점수가 올라갑니다.
- **레벨업**: 점수가 오를수록 낙하 속도가 빨라집니다.
